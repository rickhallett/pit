## Implementation Notes

- Each preset = bout config file with:
  - `preset_id`: string
  - `name`: display name
  - `description`: one-liner for picker
  - `agents`: array of agent configs
  - `turn_order`: sequential by default, could randomize
  - `max_turns`: varies by tier (see MEMORY.md)
  
- Agent config:
  - `agent_id`: string
  - `name`: display name
  - `system_prompt`: the prompt above
  - `color`: hex for UI differentiation
  - `avatar`: emoji or asset key

- **Gloves Off** is the only preset with user input (topic). Build that form into the picker.
